## Configuração inicial:
- Clonar repositório 

```
$ git clone 
$ cd chatbot-deployment
```

- Instale as Dependencias
```
>>> import nltk
>>> nltk.download('punkt')
```

- Treinar
```
$ (venv) python train.py
```

- Rodar 
```
>>> python app.py
```
