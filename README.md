## Configuração inicial:
- Clonar repositório 

```
$ git clone 
```

- Instale as Dependencias
```
>>> import nltk
>>> nltk.download('punkt')
```

- Treinar
```
$ (venv) python train.py
```

- Rodar 
```
>>> python app.py
```
